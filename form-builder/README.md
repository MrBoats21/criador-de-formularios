Passo a passo de como rodar o projeto <br> <br>
1 - clone o projeto ou faça o download do zip e extraia (Mais fácil).<br>
2 - Coloque os a pasta form-builder e os outros 2 arquivos em uma pasta.<br>
3 - Certifique-se de que o node e o npm estão instalados no computador, caso não, siga os passos em https://docs.npmjs.com/downloading-and-installing-node-js-and-npm . <br>
4 - Abra a pasta form-builder no terminal (Clique com botão direito na pasta e selecione "Abrir no terminal") .<br>
5 - Execute o comando "npm i" (Sem aspas) para instalar as dependências do projeto. <br>
6 - Execute o comando "npm run dev" (Sem aspas) para rodar o projeto.<br>
7 - Ainda no terminal, Ctrl + Clique na url que foi disponibilizada no prompt para visualizar o projeto. <br><br>
